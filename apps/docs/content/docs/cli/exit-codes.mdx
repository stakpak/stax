---
title: Exit Codes
description: CLI exit code reference
---

| Code | Meaning                             |
| ---- | ----------------------------------- |
| 0    | Success                             |
| 1    | Validation error                    |
| 2    | Build error                         |
| 3    | Registry error                      |
| 4    | Package resolution error            |
| 5    | Materialization compatibility error |
| 6    | Signature / verification error      |
