Push a built artifact to an OCI registry.

## Usage

```bash
stax push ghcr.io/myorg/agents/backend-engineer:3.1.0
stax push --all-personas ghcr.io/myorg/agents/backend-engineer
```

## Options

| Flag | Description |
| --- | --- |
| `--all-personas` | Push all persona variants |
