Scaffold a new agent or package project.

## Usage

```bash
stax init
stax init --agent
stax init --package
stax init --template github-workflow
```

## Options

| Flag | Description |
| --- | --- |
| `--agent` | Initialize an agent project |
| `--package` | Initialize a package project |
| `--template` | Use a specific template |
